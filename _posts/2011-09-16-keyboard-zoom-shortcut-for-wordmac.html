Here's how to add keyboard shortcuts for Microsoft Word's Zoom features in Word:Mac 2011.


  1. In Word: Mac 2011 go to Tools --> Customize Keyboard...

  2. Select the category 'View' on the left.

  3. In the 'Commands' list, scroll down to ViewZoomPageWidth (this is what I mostly use). Select it.

  4. Put your cursor in the box next to 'Press new keyboard shortcut:'.

  5. Press a combination of keys (i.e., your chosen shortcut. I chose Command+Option+=)

  6. If your chosen shortcut it already assigned it will be indicated below following the statement, 'Currently assigned to:'.

  7. Finally, click 'Assign'.

  8. Repeat the same operation for other zoom commands you may need.

  9. Select OK when finished.

If you're a keyboard-fiend, like me, this kind of shortcut assignment(s) can greatly speed-up your workflow. You can do this for any other menu item.
